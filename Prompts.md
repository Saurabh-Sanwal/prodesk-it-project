## My understanding

While building this project, I learned and practiced:

- How `transition` works for smooth animation effects
- How `transform` is used for hover effects like scale, rotate, and translate
- How JavaScript DOM manipulation works using `classList.toggle`
- How scroll-based animations work using `getBoundingClientRect()`
- How typing effects are created using loops and `setTimeout`

I modified and combined AI suggestions with my own experimentation in CSS and JavaScript to build and improve the final project. I also tested different values for transitions, transforms, and animations to understand how they affect UI behavior.
